Description

This lab introduces working with the ESP32 and has 2 separate tasks to complete. The first task is to build a circuit that has 3 lights that blink one after another, and the second task requires you to build a circuit that has two push buttons where one is designated to off and the other is designated to on.

Hardware components:

  • For Task 1:
    • GPIO Extension Board x1
    • ESP32-WROVER x1
    • Breadboard x1
    • Jumper wire x5
    • LED x3
    • Resistor x3 at 1 k ohm
  • For Task 2:
    • GPIO Extension Board x1
    • ESP32-WROVER x1
    • Breadboard x1
    • Jumper wire x5
    • Resistor x3 (x2 at 10 k ohms, x1 at 220 ohms)
    • LED x1
    • Push button x2
