Description

This lab introduces working with the ESP32 and has 2 separate tasks to complete. The first task is to build a circuit that has 3 lights that blink one after another, and the second task requires you to build a circuit that has two push buttons where one is designated to off and the other is designated to on.

Hardware components:

  • For Task 1:
    • GPIO Extension Board x1
    • ESP32-WROVER x1
    • Breadboard x1
    • Jumper wire x5
    • LED x3
    • Resistor x3 at 1 k ohm
  • For Task 2:
    • GPIO Extension Board x1
    • ESP32-WROVER x1
    • Breadboard x1
    • Jumper wire x5
    • Resistor x3 (x2 at 10 k ohms, x1 at 220 ohms)
    • LED x1
    • Push button x2

Task 1:

            
from time import sleep_ms
from machine import Pin

led=Pin(2,Pin.OUT)
led2=Pin(15, Pin.OUT)
led3=Pin(0, Pin.OUT)#create LED object from pin2,Set Pin2 to output
try:
    while True:
        led.value(1)        #Set led turn on
        sleep_ms(1000)
        led.value(0) 
        
        led2.value(1)        #Set led turn off
        sleep_ms(1000)
        led2.value(0)
        
        led3.value(1)        #Set led turn off
        sleep_ms(1000)
        led3.value(0)
        
        
except:
    pass

Task 2:

            
import time
from machine import Pin

led = Pin(2, Pin.OUT)
button = Pin(12, Pin.IN, Pin.PULL_UP)
button2 = Pin(13, Pin.IN, Pin.PULL_UP)

ledVal = False

while True:
    if not button.value():
        ledVal = False  # Button 1 is pressed, turn off the LED
    if not button2.value():
        ledVal = True   # Button 2 is pressed, turn on the LED
    
    # Set the LED state based on ledVal
    if ledVal:
        led.value(1)
    else:
        led.value(0)
